Changelog

Magic Laser v2.1 (May 5, 2022)

  • Added ability to import SVG files as vectors (only paths, polygons, and polylines)
  • Added ability to import DXF files as vectors
  • Added a grid template vector to help position objects on the print surface
  • Added ability to set the canvas engraver without having to connect first
  • New top bar icons
  • Text entry font selection now shows font names in their own fonts
  • Changed project savefile extension from .xj to .mag
  • Associated .dxf and .svg files with the app
